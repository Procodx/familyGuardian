# FamilyGuardian Mobile App

Android application skeleton built with Kotlin.

## Modules Structure

- **activities/**: UI Screens (Guardian Mode, Panic Trigger).
- **services/**: Background operations (Location tracking, Battery monitoring).
- **utils/**: Helper classes for hardware access (GPS, Audio).
- **models/**: Data entities.
- **network/**: Data sync and API interaction placeholders.
- **firebase/**: Firebase SDK integration placeholders.

## To Get Started

1. Open this folder in Android Studio.
2. Sync Gradle dependencies.
3. Add `google-services.json` to the `app/` directory once Firebase is configured.
